# RPS (Relative Price Strength) 量化選股工具

這是一個基於 Python 的股價相對強度 (Relative Price Strength, RPS) 計算與篩選工具。RPS 是量化與趨勢交易中常用的動量指標，用於識別市場中的強勢股。

## 核心概念

**RPS (股價相對強度)** 的核心是將個股在指定週期（如 120 日或 250 日）的漲幅放到全市場進行橫向排名，換算成 0–100 分。

- **含義**：RPS = 90 表示該股在過去 N 日的漲幅跑贏全市場 90% 的標的。
- **本質**：相對收益排名，而非絕對漲幅。能識別「弱市抗跌、強市領漲」的真強勢股。

## 功能特點

1.  **雙週期 RPS 計算**：同時計算 **120 日（中期）** 與 **250 日（長期）** 的 RPS 值。
2.  **強勢股篩選**：
    - **高 RPS 門檻**：篩選 RPS_120 和 RPS_250 均大於等於 80 的股票。
    - **趨勢確認**：檢查 RPS 是否處於上升趨勢（當前 RPS > 10 日前 RPS），確保「強者恆強」。
3.  **自動化數據獲取**：使用 `yfinance` 下載美股數據（預設包含 S&P 500 主要成分股）。

## 安裝與環境

本項目建議使用 [uv](https://github.com/astral-sh/uv) 進行依賴管理與執行。

```bash
# 安裝依賴
uv sync
```

## 使用方法

1.  確保已安裝 `uv`。
2.  使用 `uv` 直接運行 `main.py`（會自動處理虛擬環境與依賴）：

```bash
uv run main.py
```

程式將會：

1.  從 Yahoo Finance 下載指定股票池的歷史股價數據。
2.  計算每隻股票的 RPS_120 和 RPS_250。
3.  輸出篩選後的強勢股名單（條件：RPS ≥ 80 且 處於上升趨勢）。

## 篩選邏輯詳解

本工具採用以下量化選股標準：

- **核心池**：`RPS_120 >= 80` 且 `RPS_250 >= 80`。
- **趨勢確認**：要求 `Current RPS > RPS (10 days ago)`，排除掉雖然 RPS 高但動能正在衰退的股票。
- **排序**：最終結果按 `RPS_120` 由高到低排序。

## 項目結構

- `main.py`: 主程式，包含 RPS 計算 (`calc_rps`) 和篩選邏輯 (`screen_rps`)。
- `price_data.py`: 負責定義股票池 (`TICKERS`) 並從 Yahoo Finance 下載數據。
